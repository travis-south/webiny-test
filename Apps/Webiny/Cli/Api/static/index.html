Welcome to Webiny CLI API!

This is intended for future upgrades and ideas we have regarding project management so this is just the starting point. The API server is spawned as soon as you run webiny-cli in your terminal.

Still there are a few things you can do right now:

  1. See status of your project in JSON format
  2. Run development builds: /?action=develop&app=Webiny.Core
  3. Stop development: /?action=develop.stop
  4. Develop your own API methods using CLI plugins directly from your Webiny project.
